### 数学建模（双语）
计算机学院张禹老师授课，虽名为双语，只有ppt是英语。  
会不定期学习通扫码签到。  
期末考试**开卷**，建议打印ppt带入。三四个大题，只要套ppt上面的题目格式答题分不会太低，但是想高分不太容易。
